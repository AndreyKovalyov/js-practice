4. Как получить остаточные элементы, которые мы не присвоили, в отдельную переменную?
  `("Consul", "of the Roman Republic"; width, title)`

  У массива
  `let [name1, name2] = ["Julius", "Caesar", "Consul", "of the Roman Republic"];`

  У объекта

```javascript
  let options = {
    title: "Menu",
    width: 100,
    height: 200
  };

  let { title } = options;
```
